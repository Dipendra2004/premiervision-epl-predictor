# Social Preview Design Blueprint (Canva)

Use this guide to create a GitHub social preview image for this project.

## Canvas Setup

1. Size: 1280 x 640 px
2. Background: light gradient (left to right)
3. Color stops:
   - #f4f5f7
   - #ebecf0
   - #dfe1e6

## Typography

1. Title font: Rajdhani Bold
2. Subtitle font: Rajdhani Medium
3. Accent font: Rajdhani SemiBold

## Text Content

1. Main title:
   PremierVision
2. Subtitle:
   EPL Match Outcome Predictor
3. Supporting line:
   Streamlit + TensorFlow + Real Team Stats
4. Tag chips:
   Home Win | Draw | Away Win

## Layout

1. Left side (about 60% width):
   - Big title near upper-left
   - Subtitle under title
   - Supporting line under subtitle
   - Tag chips in one row near bottom-left
2. Right side (about 40% width):
   - Place 2 team logos facing each other
   - Put VS between them
   - Add mini probability card:
     Home 52% | Draw 24% | Away 24%

## Style Direction

1. Primary blue: #0052cc
2. Accent purple: #6b38c7
3. Text dark: #172b4d
4. Muted text: #5e6c84
5. Card background: white at 85% opacity
6. Card border: #dfe1e6

## Polish

1. Add soft shadow under title and cards.
2. Keep generous spacing around content.
3. Export as PNG (high quality).

## GitHub Upload Path

1. Open repo on GitHub.
2. Go to Settings > General.
3. Scroll to Social preview and upload the exported PNG.
